# Neon District

Neon District — атмосферная RPG в стиле киберпанк с открытым миром, имплантами и системой фракций.

## Особенности

- Огромный ночной мегаполис
- Улучшения тела и нейроимпланты
- Система репутации
- PvP арены
- Кооперативные миссии

## Игровые фракции

| Фракция | Описание |
|----------|------------|
| SYN Corp | Корпорация технологий |
| GhostWire | Хакеры и подполье |
| Iron Dogs | Наемники |

## Минимальные требования

- RAM: 8 GB
- GPU: GTX 1060
- Storage: 40 GB

## Запуск

```bash
npm install
npm run start
```
