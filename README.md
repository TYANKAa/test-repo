# TaskFlow

TaskFlow — это простой сервис для управления задачами, заметками и командной работой.

## Возможности

- Создание задач
- Назначение исполнителей
- Система тегов
- Markdown заметки
- Уведомления
- REST API

## Технологии

- Node.js
- Express
- MongoDB
- React
- Docker

## Установка

```bash
git clone https://github.com/example/taskflow.git
cd taskflow
